This section organizes my working notes on environment setup, computer vision, robotics debugging, reinforcement learning, and hands-on training. The goal is not only to store scattered records, but to turn them into reusable technical documentation.

## How To Browse

- Start with `Environment` if you want a stable workstation, Python environment, WSL setup, or Git workflow.
- Start with `Computer Vision` if you want dataset, annotation, YOLO, or workflow notes.
- Start with `Robotics` if you are debugging RealSense, Jetson, or embedded vision setups.
- Start with `Fundamentals` if you want quick explanations of API, tensors, broadcasting, and DOF.
- Start with `RL and Simulation` if you are exploring Isaac Lab or simulator-side workflows.
