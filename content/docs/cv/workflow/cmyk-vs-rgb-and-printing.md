---
title: CMYK、RGB 与印刷基础
---

CMYK 是一种**减色模式**（Subtractive Color Model），主要用于**印刷行业**。它代表四种基本油墨颜色：

- **C** = **Cyan**（青色）
- **M** = **Magenta**（品红色/洋红）
- **Y** = **Yellow**（黄色）
- **K** = **Key Plate**（定位套版色，通常指黑色 Black）

### 核心原理

CMYK 模式基于光线被纸张吸收和反射的原理。

1. **减色法**：与屏幕显示（RGB，加色法，光越叠加越亮）不同，CMYK 是通过油墨吸收光线来呈现颜色的。理论上，青色、品红和黄色混合应该产生黑色，但在实际印刷中，这三种颜色混合往往产生深褐色或浑浊的灰色，且成本高、干燥慢。
2. **K (Black) 的作用**：为了解决上述问题并增强图像的对比度和细节，印刷中引入了纯黑色油墨（K）。"K"代表"Key"，因为在传统四色印刷中，黑色版通常作为定位套准的关键版（Key Plate），其他颜色都围绕它对齐。

### 主要应用场景

- **彩色印刷**：杂志、报纸、宣传册、海报、包装盒等所有使用物理油墨打印的材料。
- **喷墨/激光打印机**：家用和商用打印机通常使用 CMYK 墨盒或碳粉。

### 与 RGB 的区别

| 特性         | RGB                         | CMYK                               |
| ------------ | --------------------------- | ---------------------------------- |
| **全称**     | Red, Green, Blue            | Cyan, Magenta, Yellow, Key (Black) |
| **模式类型** | 加色模式 (光)               | 减色模式 (颜料/油墨)               |
| **主要用途** | 屏幕显示 (手机、电脑、电视) | 物理印刷 (纸张、布料等)            |
| **色彩范围** | 色域较广，色彩鲜艳          | 色域相对较窄，某些亮色无法完美还原 |
| **混合结果** | 全叠加为白色                | 全叠加为黑色 (理论上)              |

### 注意事项

如果你在设计用于打印的文件（如名片、海报），务必将颜色模式设置为 **CMYK**。如果直接使用 **RGB** 模式设计并送去印刷，成品颜色往往会比屏幕上看到的暗淡许多，出现明显的色差。
