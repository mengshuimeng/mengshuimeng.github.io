# 8.开始标注

### 0 首先输入法要在英文输入状态

### 1 因为我们有四个标签

所以不管图像是什么，我们都要先按照顺序标出四个假标签![屏幕截图 2025-11-19 191501](./assets/屏幕截图 2025-11-19 191501.png)

保存

然后打开 开始序号\txt\classes.txt  检查是不是有四个

![屏幕截图 2025-11-19 191538](./assets/屏幕截图 2025-11-19 191538.png)

然后删掉我们刚刚的四个假标签 鼠标放在框框上右键 

![屏幕截图 2025-11-19 191803](./assets/屏幕截图 2025-11-19 191803.png)

删掉后正常标注

标注完后再检查txt是不是我们需要的数字



### 2 画标记

按下w快捷键  或者 图中标黄的Create

![屏幕截图 2025-11-19 192045](./assets/屏幕截图 2025-11-19 192045.png)

然后在物体左上角按住鼠标左键 长按鼠标左键 到物体右下角松开

标注尽量”相切“图形

你能分得清的就标出框框，自己分不清的可以先随便标注一个然后保存后再删除标签再保存
