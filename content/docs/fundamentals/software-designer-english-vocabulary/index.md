## 八、备考建议

### 1. 优先记高频词

优先掌握这些高频词：
- protocol
- database
- memory
- stack
- array
- file
- cache
- CPU
- API
- DBMS
- HTML
- HTTP
- IP
- TCP
- UDP
- SQL
- URL
- XML

### 2. 按专题记忆

可以按下面分类背：
- **硬件类**：CPU、RAM、ROM、cache、main board、adapter
- **网络类**：HTTP、FTP、TCP、UDP、IP、ARP、DNS、LAN、PSTN
- **数据库类**：database、DBMS、SQL
- **软件工程类**：protocol、template、system software、Programming language
- **多媒体与显示类**：pixel、video、LCD、CRT、VRR、VOD

### 3. 记忆方法

- 英文缩写 + 全称 + 中文意思，一起记
- 单词不要孤立背，尽量结合真题语境背
- 每天滚动复习 20～30 个，比一次性猛背有效
